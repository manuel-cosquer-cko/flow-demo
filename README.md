 ## Demo Flow
 
### Prerequites

* This application requires at least PHP 8.0
* This application requires [Composer](https://getcomposer.org/)
* This application requires [Symfony CLI](https://symfony.com/download/)

### Launch

* Run `composer install` for downloading the dependencies
* Run `symfony server:ca:install` for installing the certificate authority (run this only once on your machine)
* Run `symfony server:start` for running a local HTTP server
* Go on the link displayed on the console, can be [localhost:8000](https://localhost:8000/) for example (use `localhost` instead of `127.0.0.1`)


### Notes

* Add your own tokens in the `.env` file.
